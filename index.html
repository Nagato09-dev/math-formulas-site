CURVA DO CORAÇÃO

Equação Implícita

(x² + y² - 1)³ - x²y³ = 0

Equação Paramétrica

x = 16sen³(t)

y = 13cos(t) - 5cos(2t) - 2cos(3t) - cos(4t)

Cardioide (Polar)

r = 1 - sen(θ)

FÓRMULAS DE PI (π)

Definição Geométrica

π = C / d

Série de Leibniz

π/4 = 1 - 1/3 + 1/5 - 1/7 + ...

Fórmula BBP

Σ (1/16^k) * [4/(8k+1) - 2/(8k+4) - 1/(8k+5) - 1/(8k+6)]

Fórmula de Ramanujan

1/π = (2√2/9801) * Σ [(4k)!(1103+26390k) / (k!^4 * 396^(4k))]
